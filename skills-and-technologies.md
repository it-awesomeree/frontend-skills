# Skills & Technologies

Tech stack used in the AWESOMEREE Web App.

## Frontend
- **React 19** + **Next.js 15** (App Router)
- **TypeScript 5** (99.5% of codebase)
- **Tailwind CSS 3** + shadcn/ui (Radix UI primitives)
- Framer Motion (animations)
- Chart.js + Recharts (data viz)
- React Hook Form + Zod (forms & validation)
- Lucide React (icons)
- Sonner (toast notifications)
- next-themes (dark/light mode)

## Backend
- Next.js API Routes (REST)
- MySQL2 (primary DB) + PostgreSQL/Supabase (secondary DB)
- Firebase Admin (auth & services)
- Google Generative AI / Gemini
- Speakeasy (2FA/TOTP)
- Custom middleware for auth + RBAC

## Data & Reporting
- jsPDF + PDFKit (PDF generation)
- XLSX (Excel handling)
- CSV exports

## DevOps
- Google Cloud Platform (App Engine)
- GitHub Actions (CI/CD - 7-agent swarm pipeline)
- ESLint 9
- Multi-environment setup (production + test)

## API Integrations
- Shopee API (MY & SG regions)
- OpenAI API
- Google Gemini AI
- Firebase Auth + Google OAuth
